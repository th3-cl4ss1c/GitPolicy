# Правила Commit/Push (минимум бюрократии)

## Коммиты

- Сообщения коммитов на русском.
- Формат:

```text
<что было не так>
<что изменено>
```

- Рекомендуется добавлять:
- `Проверка:`
- `Риски:`

## Push

- Перед push выполняется лёгкая проверка: `bash -n run.sh`.
- Если нужно пропустить локальный `pre-push` хук: `SKIP_PRE_PUSH=1 git push ...`.
